1. Unzip the Assignment2.zip file
2. In the unzipped location there will be separate folders for each program
3. Go to each folder and edit the variables in the Makefile to run the program for your configuration
4. In particular, local input and output locations will have to be modified. The AWS variables and input output locations will have to be edited.
5. After making changes to the MakeFile, open the terminal from each of the program directories and execute "make alone" to run the program in your local land "make cloud" to run it in AWS
6. This will create the output files in the directory mentioned in the Makefile
